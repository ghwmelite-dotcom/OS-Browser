Encrypted

Your messages, your keys

End-to-end encrypted messaging built for Ghana's civil service. The server never sees your plaintext.

ECDH P-256 + AES-GCM 256-bit

Military-grade encryption using elliptic curve Diffie-Hellman key exchange and AES-GCM symmetric encryption.

.gov.gh email verification

Only verified government email holders can access the messenger. Identity tied to your civil service credentials.

Server never sees plaintext

Messages are encrypted on your device before transmission. The server only handles encrypted blobs.

Real-time WebSocket

Instant message delivery via WebSocket connections. No polling. No delays. Real-time conversations.
