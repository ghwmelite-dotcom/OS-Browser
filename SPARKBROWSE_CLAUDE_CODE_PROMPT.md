# Superseded

This document has been superseded by the **Baobab** design spec.

The original SparkBrowse build prompt described a web-app proxy browser that
would have hit `X-Frame-Options` walls on most major sites. After
brainstorming on 2026-05-01, the product was reframed as **Baobab — The
African AI Browser**: a real standalone browser (Tauri desktop + Expo mobile,
Cloudflare backend) with sovereignty as an engineered feature rather than a
slogan.

→ See `docs/superpowers/specs/2026-05-01-baobab-browser-design.md`
